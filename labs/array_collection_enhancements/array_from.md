The `Array.from()` method creates a new, shallow-copied Array instance from an array-like or iterable object.

```js
/*
Use Array.from to create an array from a string
*/
const str = 'hello';
const arr = Array.from(str);
console.log(arr);
```

```solution
const str = 'hello';
const arr = Array.from(str);
console.log(arr);
```
